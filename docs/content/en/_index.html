{{< /blocks/cover >}} {{% blocks/lead color="secondary" %}} What is Redis Operator? A Golang based redis operator that will make/oversee Redis standalone and cluster mode setup on top of the Kubernetes. It can create a redis cluster setup with best practices on Cloud as well as the Bare metal environment.
